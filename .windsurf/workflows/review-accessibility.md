# Accessibility & UX Review

1. Check all interactive elements have proper labels and aria attributes
2. Verify keyboard navigation works (tab order, enter/space to activate)
3. Review color contrast ratios for text readability (dark/light mode)
4. Check loading states - all async operations show spinners/skeletons
5. Verify responsive design - test at mobile, tablet, desktop breakpoints
6. Review toast notifications - are they informative and non-intrusive
7. Check file upload UX - drag-and-drop support, file size feedback, progress indication
